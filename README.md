# Movie-Night-"Skip-the-Scrolling"
Movie Night is a Python project that helps people decide what to watch without spending ages scrolling through streaming services.

The idea is simple: the user answers a few questions about what they're in the mood for, and the program recommends a movie or TV show that matches those preferences. As the project develops, it will also be able to understand free-text input, keep track of movies the user has already watched, and make more personal recommendations over time.

I'm building this project to learn more about Python, pandas, and recommendation systems while making something that people might actually find useful.

## Current Features

- Genre-based recommendations
- Multiple genres per movie
- Case-insensitive matching

## Planned Features

- Mood-based recommendations
- Watch history
- Random recommendations
- Streamlit interface
- More personalized recommendations
